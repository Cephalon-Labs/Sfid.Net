# Sfid.Net Logo Assets

## Files

- `sfid-logo-source.svg`
  The editable master logo source. This is the main file to adjust colors, spacing, and typography.
- `sfid-mark.svg`
  The icon-only mark for avatars, package badges, and square placements.

## Design Intent

The logo combines:

- a hexagonal network frame to suggest clustered/distributed systems
- linked node endpoints to hint at datacenter and worker coordination
- a bold central `S` ribbon to keep the mark recognizable at small sizes

## Palette

- Midnight: `#0F172A`
- Slate: `#1E293B`
- Sky: `#38BDF8`
- Teal: `#14B8A6`
- Amber: `#F59E0B`
- Soft Sand: `#FFF7ED`

## Editing Notes

- Both assets are plain SVG and can be edited in Figma, Illustrator, Inkscape, or any text editor.
- The primary logo keeps the wordmark as editable `<text>` nodes so the naming can be refined later without redrawing paths.
- If you need a locked production export for print, open `sfid-logo-source.svg` in a vector editor and convert the text to outlines before export.
